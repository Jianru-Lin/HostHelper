我经常会遇到需要修改 host 文件的情况。
我通常会使用记事本（Notepad）打开 host 文件，然后编辑并保存。
每次我都需要以管理员权限启动记事本，然后定位到 host 文件所在的目录（例如 c:\windows\system32\drivers\etc\）下，选择打开，编辑后再保存。
一次两次还好，如果经常需要这么做，那真是太麻烦了。
所以我花了几分钟的时间写了这个简单得不能再简单的工具，你只要双击运行它，它就会自动找到 host 文件，并读取出其中的内容。你可以编辑，然后点击“保存”按钮，所有的改动就被保存下来了。
真的很方便。


注意：Windows 7 的用户可以直接运行本程序，如果是较低版本的 Windows 用户，可能需要安装 .Net framework （3.5及以上版本，建议安装4.0版）才能运行。
